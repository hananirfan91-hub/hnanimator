# Heart Name Animator

A dark-themed single page site that shows a beating heart, allowing you to input your name and see advanced hover animations. Perfect for TikTok video creation!

## Features
- **Aesthetic Dark Theme:** Deep black background with neon ambient glows.
- **Interactive Heart:** Beating heart that disappears when you type.
- **Dynamic Name Animation:** Staggered hover animations with TikTok colors (Cyan/Pink).
- **Studio Mode:** Hide UI for clean screen recording.
- **Sound Effects:** Satisfying typing and hover sounds.
- **Social Sharing:** Easily share to Twitter, WhatsApp, or copy link.

## Deployment on Vercel

This project is a Vite-based Single Page Application (SPA). To deploy it on Vercel:

1. **Push to GitHub:** Push this code to a GitHub repository.
2. **Import to Vercel:** Go to [vercel.com](https://vercel.com) and import your repository.
3. **Auto-Detection:** Vercel will automatically detect the Vite framework.
4. **Environment Variables:** If you use the Gemini API features, add `GEMINI_API_KEY` to your Vercel project settings.
5. **Build & Deploy:** Click "Deploy".

The `vercel.json` file is already included to handle SPA routing and clean URLs.
